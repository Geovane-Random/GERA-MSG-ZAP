# GERA-MSG-ZAP

Projeto front-end simples — arquivos iniciais enviados para o repositório.

## Visão geral

Este repositório contém arquivos iniciais de um projeto front-end:

- `index.html` — página principal
- `script.js` — lógica do cliente
- `style.css` — estilos
- `config.json` — configurações

## Como usar

1. Abra `index.html` no navegador.
2. (Opcional) Sirva localmente com um servidor estático, por exemplo:

```bash
npx serve .
```

## Observações

Edite `config.json` conforme necessário antes de executar a aplicação.
